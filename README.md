## Шаг 9. Обработка ошибок API

**Задание:**
VK API не всегда отвечает успешно. Если что-то пошло не так (невалидный токен, неправильная ссылка), в JSON-ответе вместо ключа `response` будет ключ `error`. Добавь в свою функцию проверку через исключение, через условный оператор `if`: если в ответе есть ошибка — сообщи об этом, подняв исключение.

**Вопросы к тебе:**

- Что такое `raise` и чем оно отличается от `print`? Зачем поднимать исключение, а не просто печатать ошибку?
- Как выглядит объект ошибки в ответе VK API? Где внутри него лежит текст ошибки?

**Документация:**
- Исключения в Python: https://docs.python.org/3/tutorial/errors.html
